# Виртуальный датчик для контроля процесса обжига

## Описание проекта

Цель проекта — разработка модели машинного обучения (виртуального датчика), позволяющей прогнозировать концентрацию продукта в печи на основе телеметрических данных.

## Данные

- `data_train.csv` — телеметрия (минутные данные)
- `target_train.csv` — лабораторные замеры (нерегулярные)
- `data_test_small.csv` — тестовая телеметрия
- `target_test_small.csv` — тестовые замеры

## Этапы работы

### 1. EDA
- анализ телеметрии
- анализ целевой переменной
- учет задержки лабораторных измерений

### 2. Feature Engineering
- лаговые признаки
- скользящие статистики (mean, std, min, max)
- производные (diff)

### 3. Моделирование
Были обучены модели:
- Linear Regression
- Random Forest
- Gradient Boosting

### 4. Оценка качества
Использованы метрики:
- MAE
- RMSE
- MAPE
- WAPE
- Direction Accuracy

Также проведены:
- анализ остатков
- анализ важности признаков
- визуализация прогнозов

## Результаты

Лучшая модель — Gradient Boosting.

Она показала:
- наименьшую ошибку
- хорошее соответствие динамике процесса
- разумное время обучения
