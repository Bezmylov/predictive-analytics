# Прогнозирование температуры на основе временных рядов

## Описание проекта
Проект посвящён прогнозированию температуры воздуха на горизонте 30 дней с использованием методов машинного обучения.

Решение реализует двухэтапный подход:
1. Классификация временных рядов (определение структуры климатических данных)
2. Прогнозирование температуры (регрессионные модели)

## Используемые технологии

- Python
- pandas, numpy
- scikit-learn
- sktime (ROCKET)
- statsmodels (статистические тесты)
- matplotlib, seaborn (визуализация)

## Данные

Используются временные ряды погодных параметров:
- температура
- влажность
- осадки
- ветер
- давление

Данные представлены в формате `.parquet` по городам.
